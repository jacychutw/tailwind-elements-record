# Tailwind Elements 介紹與實作
> Tailwind CSS 是一個開放原始碼 CSS 框架。創建「工具程式型（Utility）」CSS 類別列表。<br>
> Tailwind Elements 是非 Tailwind CSS 官方團隊，將 Bootstrap 組件使用 Tailwind CSS 重新創建。<br>
> 此專案多著墨在 CSS 編輯，故功能不齊全。<br>


📝 &nbsp; 專案教學文件：

🚀 &nbsp; 快速開始：<br>
- 安裝相依套件
```
npm install
```
- 編輯input.css
- 啟動開發伺服器
```
npm run build-css
```
